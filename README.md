# Доброе Информ.Бюро — медиа-волонтёры Кемского округа

Одностраничный сайт для медиа-волонтёров Движения Первых Кемского муниципального округа.

## Что внутри

- описание проекта «Доброе Информ.Бюро»;
- роли медиа-волонтёров;
- алгоритм работы перед публикацией;
- календарь на июль 2026 года;
- медиаплан;
- чек-листы и шаблоны для работы;
- ссылка на вакансию на Добро.рф.

## Ссылка на вакансию

https://dobro.ru/event/11788822

## Публикация сайта

Сайт — статический. Главная страница находится в файле `index.html`.

Для GitHub Pages можно использовать публикацию из ветки `main` или workflow `.github/workflows/pages.yml`.
