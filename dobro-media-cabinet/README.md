# Добро.Медиа — кабинет медиа-волонтёра

Это новая версия сайта: интерактивный календарь заданий, видимость занятых активностей и админ-панель для Кустова Евгения Валерьевича.

## Что будет у ребят

- календарь на июль;
- карточка задания по каждой активности;
- кнопка «Взять активность»;
- кнопка «Сдать материал»;
- видимость, кто уже взял задание.

## Что будет у администратора

- страница `/admin`;
- изменение статуса активности;
- учёт времени;
- комментарии администратора;
- живое обновление данных через Supabase.

## Стек

- Next.js;
- Supabase;
- Vercel;
- GitHub.

## Следующие шаги

1. Создать проект Supabase.
2. Выполнить SQL из `supabase/schema.sql`.
3. Создать проект Vercel с корневой папкой `dobro-media-cabinet`.
4. Добавить переменные окружения из `.env.example`.
5. Открыть сайт и проверить календарь.
